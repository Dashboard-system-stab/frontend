# Инструкция 3

## Структура проекта

### app.py — точка входа
Основной файл запуска. Собирает приложение из модулей, регистрирует колбэки, определяет вкладки интерфейса. Здесь же инициализируются WebSocket подключения к бэкендам.

### graph.py — график и панель данных
Содержит лейаут и логику первой вкладки:
- Отрисовка графика температуры в реальном времени
- Таблица с датами и значениями температуры
- Поля ввода коэффициентов X и Y и кнопка отправки на бэкенд
- Приём данных от бэкенда Романа по WebSocket (`ws://localhost:8765`)

### editor.py — редактор скриптов
Содержит лейаут и логику второй вкладки:
- Выпадающий список доступных `.py` файлов
- Текстовый редактор для просмотра и редактирования кода
- Кнопки "Открыть" и "Сохранить"
- Общение с бэкендом Ивана по WebSocket (`ws://localhost:7777`)

### styles.py — стили
Словари стилей для элементов интерфейса (кнопки, поля ввода, панели). Вынесены в отдельный файл для удобства.

## Что реализовано по новому ТЗ

### Приём данных от бэкенда
- Настроено получение конвертированных данных от бэкенда Романа в формате `{"date": "...", "temperature": ...}`
- Данные отображаются на графике в реальном времени через очередь — по одному значению за тик

### UI редактора скриптов
Разработана вторая вкладка с текстовым редактором Python-кода:

**Функция "Открыть скрипт"**
- По нажатию кнопки "Открыть" фронтенд отправляет запрос `{"type": "get_list"}` бэкенду Ивана
- Иван возвращает список доступных `.py` файлов
- Список отображается в выпадающем меню
- При выборе файла отправляется запрос `{"type": "get_file", "filename": "..."}` и содержимое файла загружается в редактор

**Функция "Сохранить скрипт"**
- По нажатию кнопки "Сохранить" текст из редактора отправляется бэкенду в формате `{"type": "save_file", "filename": "...", "content": "..."}`
- Иван перезаписывает файл на сервере и возвращает `{"status": "ok"}`
- На экране появляется надпись "Успешно сохранено"

## Формат обмена данными

| Направление | Формат |
|---|---|
| Роман → Владислав (данные) | `{"date": "2024-01-01", "temperature": 20.0}` |
| Владислав → Роман (коэффициенты) | `{"type": "coefficients", "x": 2.0, "y": 2.0}` |
| Владислав → Иван (список файлов) | `{"type": "get_list"}` |
| Иван → Владислав (список файлов) | `{"files": ["script1.py", "script2.py"]}` |
| Владислав → Иван (чтение файла) | `{"type": "get_file", "filename": "script1.py"}` |
| Иван → Владислав (содержимое) | `{"filename": "script1.py", "content": "..."}` |
| Владислав → Иван (сохранение) | `{"type": "save_file", "filename": "script1.py", "content": "..."}` |
| Иван → Владислав (подтверждение) | `{"status": "ok"}` |
